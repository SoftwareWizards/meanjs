A user should have the history of all their payments.

Thus, there should be a payment schema that links to the user schema

The payment schema should look something like:
Payments {
Username: string; /* This uniquely identifies the user */
Amount: string;
Type: {cash, gift card, mail check, in person, western union, paypal}
Payment date: string;
}
